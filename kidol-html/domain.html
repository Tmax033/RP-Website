This research was concluded to develop a mobile application for a toy store that enables users to interact with advanced AR features while exploring toy products and machine learning-based recommendation systems with AI-powered intelligent chatbot. While researching the previous and exsisting similar systems we were able to identify some currently available applications

This revolutionary approach was made into a mobile app for Windows and Android. “Pseudo Eye” is meant to take photos inside stores and obtain product details after a comparison. They employed CBIR (Content-based image retrieval) technology to display relevant product images from the database.

An Android app designed for shopping malls. This application's major goal is to promote consumer contact. Using AR labels to display discounts and shop names. Customers can scan a QR code to view product details after visiting a physical store.

A shopping mall navigation system. This system uses a mobile phone camera to capture real-time camera input and display the path on a map. Positions give the right direction to the user using AR and Bluetooth beacons.

Mobile application that was developed for manipulating 3D objects and deploy them in the real world. To calculate the accurate position for deploying 3D objects, accelerometer and gyroscope sensors were used in mobile phones.
